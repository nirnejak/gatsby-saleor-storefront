## 🚀 Quick start

1.  **Clone the repo.**

    Use the Gatsby CLI to create a new site, specifying this repo.

    ```sh
    # create a new Gatsby site using the default starter
    gatsby new site-folder-name https://github.com/inkoop/gatsby-starter-codedrips
    ```

2. **Available Scripts**

    ```sh
    npm start
    ```
    ```sh
    npm run build
    ```
    ```sh
    npm run format
    ```
    ```sh
    npm run lint
    ```
    ```sh
    npm run lint-fix
    ```
